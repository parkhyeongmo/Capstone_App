### 프로젝트 상세 내용


#### 사용 기술
언어: Kotlin  
프레임워크: AndroidStudio  
구동환경: Android 5.0 이상  

#### 협업 관리 도구
- Git
- Notion

사진을 이용한 **선박 및 해양플랜트 부품 품질 검사 서비스** 입니다.

---

#### 핵심 기능

1. 사진을 이용한 불량 유무 및 유형 검사 기능 
2. 이전 검사 내역을 통한 상세 정보 확인 기능
3. 사용자 - 담당자 간 통화 연결 기능
4. 부품 별 재고 관리 기능

#### 담당 업무

- UI/UX 구축
- Kakao 소셜로그인 연동
- Retrofit2를 이용한 서버와의 RESTful API 연동
- 그 외 전반적인 어플리케이션 기능 구현
